Abstract
Score- and feature-based methods are the two main ones for estimating a forensic likelihood ratio (LR) quantifying the strength of evidence. In this forensic text comparison (FTC) study, a score-based method using the Cosine distance is compared with a feature-based method built on a Poisson model with texts collected from 2,157 authors. Distance measures (e.g. Burrows’s Delta, Cosine distance) are a standard tool in authorship attribution studies. Thus, the implementation of a score-based method using a distance measure is naturally the first step for estimating LRs for textual evidence. However, textual data often violates the statistical assumptions underlying distance-based models. Furthermore, such models only assess the similarity, not the typicality, of the objects (i.e. documents) under comparison. A Poisson model is theoretically more appropriate than distance-based measures for authorship attribution, but it has never been tested with linguistic text evidence within the LR framework. The log-LR cost (Cllr) was used to assess the performance of the two methods. This study demonstrates that : (1) the feature-based method outperforms the score-based method by a Cllr value of ca. 0.09 under the best-performing settings and ; (2) the performance of the feature-based method can be further improved by feature selection.

Les méthodes basées sur le score et les caractéristiques sont les deux principales méthodes d'estimation d'un ratio de vraisemblance (LR) médico-légal quantifiant la force des preuves. Dans cette étude de comparaison de textes médico-légaux (FTC), une méthode basée sur des scores utilisant la distance cosinus est comparée à une méthode basée sur des caractéristiques construite sur un modèle de Poisson avec des textes recueillis auprès de 2 157 auteurs. Les mesures de distance (par exemple, le delta de Burrows, la distance en cosinus) sont un outil standard dans les études d'attribution de paternité. Ainsi, la mise en œuvre d'une méthode basée sur des scores utilisant une mesure de distance est naturellement la première étape pour estimer les LR pour des preuves textuelles. Cependant, les données textuelles enfreignent souvent les hypothèses statistiques sous-jacentes aux modèles basés sur la distance. De plus, de tels modèles évaluent uniquement la similitude, et non le caractère typique, des objets (c'est-à-dire des documents) comparés. Un modèle de Poisson est théoriquement plus approprié que les mesures basées sur la distance pour l'attribution de la paternité, mais il n'a jamais été testé avec des preuves textuelles linguistiques dans le cadre de la LR. Le coût log-LR (Cllr) a été utilisé pour évaluer les performances des deux méthodes. Cette étude démontre que : (1) la méthode basée sur les caractéristiques surpasse la méthode basée sur les scores d'une valeur Cllr d'environ 0,09 dans les paramètres les plus performants et ; (2) les performances de la méthode basée sur les caractéristiques peuvent être encore améliorées par la sélection de caractéristiques.
