Abstract
We describe the system developed by the DFKI-TalkingRobots Team for the CODI-CRAC 2021 Shared-Task on anaphora resolution in dialogue. Our system consists of three subsystems : (1) the Workspace Coreference System (WCS) incrementally clusters mentions using semantic similarity based on embeddings combined with lexical feature heuristics ; (2) the Mention-to-Mention (M2 M) coreference resolution system pairs same entity mentions ; (3) the Discourse Deixis Resolution (DDR) system employs a Siamese Network to detect discourse anaphor-antecedent pairs. WCS achieved F1-score of 55.6 % averaged across the evaluation test sets, M2 M achieved 57.2 % and DDR achieved 21.5 %.
